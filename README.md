# LLMSalon

LLMSalon 使用大语言模型进行群聊


## 运行

1. 确保已安装Python 3.13+
2. 克隆本项目：
   ```bash
   git clone https://github.com/yourusername/LLMSalon.git
   cd LLMSalon
   ```
3. 安装依赖：
   ```bash
   pip install .
   ```

4. 运行:
   ```
   gradio src/interface.py
   ```

## 配置

编辑 `src/settings.yaml` 可配置模型及prompt。
